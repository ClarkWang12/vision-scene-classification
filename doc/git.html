To run the program:
  • First run make.m inside the matlab directory of liblinear-1.96 and libsvm-3.20.
  • The "scene_categories" folder must be copied into a folder "data", which has to be created; that is, the script expects this path: "data/scene_categories/SUB-CATEGORY-FOLDER"
  • Set the required values in the following variables (the currently enabled values point to our best performing result) params.dictionarySize, params.kernel, params.method, params.dictionaryType in master.m.
  • Then, just execute master.m which would execute the require steps and provide mean accuracy and the confusion matrix.
  • All code modules were written and executed in Matlab on MAC OSX.
