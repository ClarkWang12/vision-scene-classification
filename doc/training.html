Classification Pipeline

The scene classification pipeline consists of the following steps:
  • Generate SIFT features for all the images
  • Create a random permutation of the file names in each class and pick 100 images in each class for training and use the rest of the images for testing
  • Building dictionary using:
    1. Codebook generated by K-means clustering, or,
    2. Incremental codebook optimization on the K-means codebook (Algorithm 4.1 from J. Wang et. al)
  • Encoding:
    1. Hard assignment - Histogram pyramids generated by SpatialPyramid code: Baseline Implementation and Results
    2. Soft assignment - LLC + max pooling: LLC Implementation and Results
  • SVM training:
    1. Baseline: Histogram intersection kernel is used for training SVM
    2. LLC: linear kernel is used for training SVM
  • Classify the test images using the two SVM methods
  • Create the corresponding confusion matrices
